# 创建数据表

## 创建表的基本格式

语句格式:

CREATE TABLE 表名 (

        列名     数据类型(长度)    完整性约束条件,

        列名     数据类型(长度)    完整性约束条件,
        ......
  

) engine=myisam default charset=utf8

 

**注意事项:**

   **列名、数据类型、长度是必有的； 完整性约束条件可以没有**

   **最后的 utf8 没有 -**
